# @template/pkce

pkce package for Axon AI platform.

## Installation

```bash
pnpm add @template/pkce
```

## Usage

```typescript
import { /* exports */ } from '@template/pkce';
```

## Development

```bash
# Build the package
pnpm build

# Watch for changes
pnpm dev

# Type checking
pnpm check-types

# Linting
pnpm lint
```
