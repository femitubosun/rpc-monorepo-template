# @template/db-utils

db-utils package for Axon AI platform.

## Installation

```bash
pnpm add @template/db-utils
```

## Usage

```typescript
import { /* exports */ } from '@template/db-utils';
```

## Development

```bash
# Build the package
pnpm build

# Watch for changes
pnpm dev

# Type checking
pnpm check-types

# Linting
pnpm lint
```
