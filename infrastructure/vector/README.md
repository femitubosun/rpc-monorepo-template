# @template/vector

vector package for Axon AI platform.

## Installation

```bash
pnpm add @template/vector
```

## Usage

```typescript
import { /* exports */ } from '@template/vector';
```

## Development

```bash
# Build the package
pnpm build

# Watch for changes
pnpm dev

# Type checking
pnpm check-types

# Linting
pnpm lint
```
