# @template/string-helpers

string-helpers package for Axon AI platform.

## Installation

```bash
pnpm add @template/string-helpers
```

## Usage

```typescript
import { /* exports */ } from '@template/string-helpers';
```

## Development

```bash
# Build the package
pnpm build

# Watch for changes
pnpm dev

# Type checking
pnpm check-types

# Linting
pnpm lint
```
