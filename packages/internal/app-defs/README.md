# @template/app-defs

app-defs package for Axon AI platform.

## Installation

```bash
pnpm add @template/app-defs
```

## Usage

```typescript
import { /* exports */ } from '@template/app-defs';
```

## Development

```bash
# Build the package
pnpm build

# Watch for changes
pnpm dev

# Type checking
pnpm check-types

# Linting
pnpm lint
```
