# @template/string-utils

string-utils package for Axon AI platform.

## Installation

```bash
pnpm add @template/string-utils
```

## Usage

```typescript
import { /* exports */ } from '@template/string-utils';
```

## Development

```bash
# Build the package
pnpm build

# Watch for changes
pnpm dev

# Type checking
pnpm check-types

# Linting
pnpm lint
```
