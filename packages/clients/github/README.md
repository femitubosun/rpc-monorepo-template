# @template/github

github package for Axon AI platform.

## Installation

```bash
pnpm add @template/github
```

## Usage

```typescript
import { /* exports */ } from '@template/github';
```

## Development

```bash
# Build the package
pnpm build

# Watch for changes
pnpm dev

# Type checking
pnpm check-types

# Linting
pnpm lint
```
