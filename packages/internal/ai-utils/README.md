# @template/ai-utils

ai-utils package for Axon AI platform.

## Installation

```bash
pnpm add @template/ai-utils
```

## Usage

```typescript
import { /* exports */ } from '@template/ai-utils';
```

## Development

```bash
# Build the package
pnpm build

# Watch for changes
pnpm dev

# Type checking
pnpm check-types

# Linting
pnpm lint
```
