# @template/resend

resend package for Axon AI platform.

## Installation

```bash
pnpm add @template/resend
```

## Usage

```typescript
import { /* exports */ } from '@template/resend';
```

## Development

```bash
# Build the package
pnpm build

# Watch for changes
pnpm dev

# Type checking
pnpm check-types

# Linting
pnpm lint
```
