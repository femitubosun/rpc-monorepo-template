# @template/r2

r2 package for Axon AI platform.

## Installation

```bash
pnpm add @template/r2
```

## Usage

```typescript
import { /* exports */ } from '@template/r2';
```

## Development

```bash
# Build the package
pnpm build

# Watch for changes
pnpm dev

# Type checking
pnpm check-types

# Linting
pnpm lint
```
