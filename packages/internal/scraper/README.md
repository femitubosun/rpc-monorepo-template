# @template/scraper

scraper package for Axon AI platform.

## Installation

```bash
pnpm add @template/scraper
```

## Usage

```typescript
import { /* exports */ } from '@template/scraper';
```

## Development

```bash
# Build the package
pnpm build

# Watch for changes
pnpm dev

# Type checking
pnpm check-types

# Linting
pnpm lint
```
