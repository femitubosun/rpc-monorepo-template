# @template/http

http package for Axon AI platform.

## Installation

```bash
pnpm add @template/http
```

## Usage

```typescript
import { /* exports */ } from '@template/http';
```

## Development

```bash
# Build the package
pnpm build

# Watch for changes
pnpm dev

# Type checking
pnpm check-types

# Linting
pnpm lint
```
